# 💼 Portafolio Personal – Melva Suárez

Portafolio profesional desarrollado con **React** y **Vite** para mostrar mis proyectos, habilidades y experiencia como estudiante de Desarrollo de Software en la Escuela Politécnica Nacional.

## 🚀 Tecnologías utilizadas

- React 18
- Vite
- CSS puro (sin frameworks)
- React Icons

## 📁 Estructura principal

El proyecto está organizado en componentes reutilizables:

- `Header` – Barra de navegación fija
- `Main` – Sección Hero (presentación y redes sociales)
- `About` – Información personal y tecnologías
- `Projects` – Grid de proyectos académicos
- `Contact` – Formulario de contacto simulado
- `Footer` – Enlaces rápidos y copyright

## 🧪 Cómo ejecutar en local

```bash
npm install
npm run dev
